Calunium is an Arduino clone using the ATmega644P microcontroller. The
ATmega1284P will also be supported.

Stripboard construction details are given at
http://blog.stevemarple.co.uk/2011/08/introducing-calunium-arduino-clone.html
with photographs on Flickr, http://www.flickr.com/photos/stevemarple/sets/72157627418450430/
